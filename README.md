# Haze-Ard
an application for the real-time prediction of visibility on a path/ at a place utilizing the monitored air pollution.
## Required modules
* DOWNTOWM data from VentureWithAir website
* python 3.6
* matplotlib 2.0.2
* basemap
* geopandas
* netcdf4
